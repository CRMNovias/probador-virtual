Metodología de Desarrollo

Develop: Rama de integración principal
Trabajo en Conversación: Una rama por conversación/sesión

Antes de cada desarrollo:

Arquitectura: ¿Respeta la estructura MCP?
SOLID: ¿Es extensible y mantenible?
Escalabilidad: ¿Soporta crecimiento?
Elegancia: ¿Solución limpia y simple?

Aplica YAGNI, Clean Code y KISS

Classes → PascalCase
